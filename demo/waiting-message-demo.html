Chat Widget - Waiting Message Demo

This demo showcases the new waiting message functionality. When you send a message, you'll see an animated waiting indicator that gets replaced with the actual response.

Features

  • Animated waiting dots appear when sending a message
  • Waiting message is automatically removed when response arrives
  • Works for both regular messages and widget interactions
  • Smooth transitions and professional appearance
  • Responsive design that works across all themes

Try it out!

Click the chat button in the bottom-right corner and send a message to see the waiting indicator in action.
