Progress Widget Demo

Overview

The Progress Widget renders a read-only progress indicator for showing completion status. It displays a visual bar and text indicating the current progress.

Key Features:

  • Visual progress bar
  • Customizable value and maximum
  • Percentage display
  • Custom label text
  • Read-only display

Backend Response Format

To trigger a progress widget, the backend should respond with:

{ "text": "Your download is in progress:", "widget": { "type": "progress", "label": "Download Progress", "value": 75, "max": 100, "showPercentage": true } }

Widget Data Structure

{ "type": "progress", "label": "Progress", "value": 50, "max": 100, "showPercentage": true }

Fields:

  • type: Must be "progress"
  • label: Label text for the progress bar (default: "Progress")
  • value: Current progress value (required)
  • max: Maximum value (required, must be > 0)
  • showPercentage: Show percentage in text (default: true)

Usage Examples

Download Progress

{ "type": "progress", "label": "Download Progress", "value": 75, "max": 100, "showPercentage": true }

Task Completion

{ "type": "progress", "label": "Tasks Completed", "value": 8, "max": 10, "showPercentage": true }

File Upload

{ "type": "progress", "label": "Upload Progress", "value": 256, "max": 1024, "showPercentage": true }

Simple Counter

{ "type": "progress", "label": "Steps Completed", "value": 3, "max": 5, "showPercentage": false }
