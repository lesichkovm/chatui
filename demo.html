Chat Widget Demo

This is a demo page showing how the chat widget can be embedded in any website with a single script tag.

How to Embed

Copy and paste this code into your website:

<script id="chat-widget-1" src="http://localhost:3000/chat-widget.js" data-position="bottom-right" data-color="#007bff" data-title="Chat with us"></script>

Customization Options

You can customize the widget using data attributes:

  • data-position: Widget position ('bottom-right', 'bottom-left', 'top-right', 'top-left')
  • data-color: Primary color for buttons and user messages
  • data-title: Title displayed in the chat window header

Multiple Widgets

You can add multiple widgets to the same page by using different IDs and colors:

<script id="chat-widget-1" src="http://localhost:3000/chat-widget.js" data-position="bottom-right" data-color="#007bff"></script>
<script id="chat-widget-2" src="http://localhost:3000/chat-widget.js" data-position="bottom-left" data-color="#28a745"></script>
