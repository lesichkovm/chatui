File Upload Widget Demo

Overview

The File Upload Widget renders a file upload area with drag-and-drop support. Users can upload files by dragging them or clicking to select.

Key Features:

  • Drag and drop file upload
  • Click to browse files
  • Multiple file support
  • File type filtering
  • File size validation
  • Remove files before upload

Backend Response Format

To trigger a file upload widget, the backend should respond with:

{ "text": "Please upload your documents:", "widget": { "type": "file", "label": "Upload Documents", "accept": ".pdf,.doc,.docx", "maxFiles": 3, "maxSize": 5242880, "buttonText": "Upload" } }

Widget Data Structure

{ "type": "file", "label": "Upload a file", "accept": ".jpg,.png,.pdf", "maxFiles": 1, "maxSize": 10485760, "buttonText": "Upload" }

Fields:

  • type: Must be "file"
  • label: Label text for the upload area (default: "Upload a file")
  • accept: Accepted file types (e.g., ".pdf,.doc" or "image/*")
  • maxFiles: Maximum number of files (default: 1)
  • maxSize: Maximum file size in bytes (default: 10MB)
  • buttonText: Text displayed on the upload button (default: "Upload")

Usage Examples

Document Upload

{ "type": "file", "label": "Upload Documents", "accept": ".pdf,.doc,.docx", "maxFiles": 5, "maxSize": 10485760, "buttonText": "Upload Files" }

Image Upload

{ "type": "file", "label": "Upload Profile Picture", "accept": "image/*", "maxFiles": 1, "maxSize": 5242880, "buttonText": "Upload Image" }

Any File Type

{ "type": "file", "label": "Upload File", "maxFiles": 1, "maxSize": 20971520, "buttonText": "Submit" }
