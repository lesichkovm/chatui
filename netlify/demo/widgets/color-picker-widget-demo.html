Color Picker Widget Demo

Overview

The Color Picker Widget renders a color selection interface for visual preferences. Users can pick colors using a native color picker or select from preset colors.

Key Features:

  • Native color picker
  • Preset color options
  • Real-time color preview
  • Hex color display
  • Default color support

Backend Response Format

To trigger a color picker widget, the backend should respond with:

{ "text": "Choose your preferred theme color:", "widget": { "type": "color", "label": "Select Theme Color", "defaultColor": "#007bff", "presetColors": ["#007bff", "#28a745", "#dc3545", "#ffc107", "#17a2b8"], "buttonText": "Apply" } }

Widget Data Structure

{ "type": "color", "label": "Select a color", "defaultColor": "#007bff", "presetColors": ["#ff0000", "#00ff00", "#0000ff"], "buttonText": "Submit" }

Fields:

  • type: Must be "color"
  • label: Label text for the color picker (default: "Select a color")
  • defaultColor: Default selected color in hex format (optional)
  • presetColors: Array of preset colors in hex format (optional)
  • buttonText: Text displayed on the submit button (default: "Submit")
