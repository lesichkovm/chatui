Textarea Widget Demo

Overview

The Textarea Widget renders a multi-line text input for longer responses. Users can type multiple lines of text and submit them.

Key Features:

  • Multi-line text input
  • Customizable number of rows
  • Character limit support
  • Custom placeholder text
  • Ctrl+Enter keyboard shortcut for submission

Backend Response Format

To trigger a textarea widget, the backend should respond with:

{ "text": "Please describe your issue in detail:", "widget": { "type": "textarea", "placeholder": "Describe your issue here...", "rows": 6, "maxLength": 500, "buttonText": "Submit" } }

Widget Data Structure

{ "type": "textarea", "placeholder": "Enter your response...", "rows": 4, "maxLength": 1000, "buttonText": "Submit" }

Fields:

  • type: Must be "textarea"
  • placeholder: Placeholder text for the textarea (default: "Enter your response...")
  • rows: Number of visible rows (default: 4)
  • maxLength: Maximum character count (optional)
  • buttonText: Text displayed on the submit button (default: "Submit")

Usage Examples

Feedback Form

{ "type": "textarea", "placeholder": "Please share your feedback...", "rows": 6, "maxLength": 1000, "buttonText": "Send Feedback" }

Bug Report

{ "type": "textarea", "placeholder": "Describe the bug and steps to reproduce...", "rows": 8, "maxLength": 2000, "buttonText": "Submit Report" }

Short Note

{ "type": "textarea", "placeholder": "Add a note...", "rows": 3, "buttonText": "Save" }
