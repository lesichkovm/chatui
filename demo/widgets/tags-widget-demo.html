Tags Widget Demo

Overview

The Tags Widget renders a tag input with autocomplete for multiple keywords. Users can type tags, select from suggestions, and remove tags.

Key Features:

  • Add tags by typing and pressing Enter
  • Autocomplete suggestions
  • Remove tags with × button
  • Maximum tag limit
  • Prevent duplicate tags

Backend Response Format

To trigger a tags widget, the backend should respond with:

{ "text": "What are your interests?", "widget": { "type": "tags", "label": "Select Your Interests", "placeholder": "Type and press Enter", "suggestions": ["Technology", "Sports", "Music", "Travel", "Food"], "maxTags": 5, "buttonText": "Submit" } }

Widget Data Structure

{ "type": "tags", "label": "Add tags", "placeholder": "Type and press Enter to add tag", "suggestions": ["tag1", "tag2", "tag3"], "maxTags": 10, "buttonText": "Submit" }

Fields:

  • type: Must be "tags"
  • label: Label text for the tags input (default: "Add tags")
  • placeholder: Placeholder text (default: "Type and press Enter to add tag")
  • suggestions: Array of suggested tags (optional)
  • maxTags: Maximum number of tags (default: 10)
  • buttonText: Text displayed on the submit button (default: "Submit")
