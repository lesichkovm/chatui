Rating Widget Demo

Overview

The Rating Widget renders a star or emoji-based rating system. Users can click on stars to rate something on a scale.

Key Features:

  • Customizable maximum rating
  • Star or emoji icons
  • Hover preview effect
  • Instant submission on click
  • Custom label text

Backend Response Format

To trigger a rating widget, the backend should respond with:

{ "text": "How would you rate our service?", "widget": { "type": "rating", "label": "Rate your experience", "maxRating": 5, "iconType": "stars" } }

Widget Data Structure

{ "type": "rating", "label": "Rate this", "maxRating": 5, "iconType": "stars" }

Fields:

  • type: Must be "rating"
  • label: Label text for the rating (default: "Rate this")
  • maxRating: Maximum rating value (default: 5)
  • iconType: Icon type - "stars" or "emojis" (default: "stars")

Usage Examples

5-Star Rating

{ "type": "rating", "label": "Rate your satisfaction", "maxRating": 5, "iconType": "stars" }

10-Star Rating

{ "type": "rating", "label": "Rate on a scale of 1-10", "maxRating": 10, "iconType": "stars" }

Emoji Rating

{ "type": "rating", "label": "How much do you like this?", "maxRating": 5, "iconType": "emojis" }

Try It Yourself

Type "show rating" in the chat to see the rating widget in action!

The demo simulates backend responses with rating widget data. Try hovering over stars and clicking to submit a rating.
