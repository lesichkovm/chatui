Radio Widget Demo

Overview

The Radio Widget renders radio buttons for single selection from multiple options. Users can select exactly one option and submit it.

Key Features:

  • Single selection only
  • Pre-selected option support
  • Customizable submit button text
  • Clear visual selection
  • Disable after submission

Backend Response Format

To trigger a radio widget, the backend should respond with:

{ "text": "What is your preferred contact method?", "widget": { "type": "radio", "buttonText": "Submit", "options": [ { "id": "1", "text": "Email", "value": "email" }, { "id": "2", "text": "Phone", "value": "phone" }, { "id": "3", "text": "SMS", "value": "sms" } ] } }

Widget Data Structure

{ "type": "radio", "buttonText": "Submit", "options": [ { "id": "unique-id", "text": "Display text", "value": "value", "checked": false } ] }

Fields:

  • type: Must be "radio"
  • buttonText: Text displayed on the submit button (default: "Submit")
  • options: Array of radio options
  • options[].id: Unique identifier for the option
  • options[].text: Display text for the option
  • options[].value: Value to send when selected
  • options[].checked: Pre-select the option (default: false)

Usage Examples

Contact Method

{ "type": "radio", "buttonText": "Save Preference", "options": [ { "id": "email", "text": "Email", "value": "email" }, { "id": "phone", "text": "Phone", "value": "phone" }, { "id": "sms", "text": "SMS", "value": "sms" } ] }

Frequency Selection

{ "type": "radio", "buttonText": "Set Frequency", "options": [ { "id": "daily", "text": "Daily", "value": "daily" }, { "id": "weekly", "text": "Weekly", "value": "weekly", "checked": true }, { "id": "monthly", "text": "Monthly", "value": "monthly" } ] }

Size Selection

{ "type": "radio", "buttonText": "Select Size", "options": [ { "id": "small", "text": "Small", "value": "S" }, { "id": "medium", "text": "Medium", "value": "M" }, { "id": "large", "text": "Large", "value": "L" }, { "id": "xlarge", "text": "Extra Large", "value": "XL" } ] }
