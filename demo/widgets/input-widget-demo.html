Input Widget Demo

Overview

The Input Widget renders a text input field with a submit button. Users can type text and submit it either by clicking the button or pressing Enter.

Key Features:

  • Customizable input type (text, email, number, etc.)
  • Custom placeholder text
  • Customizable submit button text
  • Enter key submission support
  • Input validation and clearing after submission

Backend Response Format

To trigger an input widget, the backend should respond with:

{ "text": "Please enter your email address:", "widget": { "type": "input", "inputType": "email", "placeholder": "Enter your email...", "buttonText": "Submit" } }

Widget Data Structure

{ "type": "input", "inputType": "text", "placeholder": "Enter your response...", "buttonText": "Submit" }

Fields:

  • type: Must be "input"
  • inputType: HTML input type (default: "text")
  • placeholder: Placeholder text for the input field
  • buttonText: Text displayed on the submit button

Supported Input Types:

  • text - Default text input
  • email - Email validation
  • number - Numeric input
  • tel - Phone number input
  • url - URL input
  • password - Password field

Usage Examples

Email Collection

{ "type": "input", "inputType": "email", "placeholder": "Enter your email address", "buttonText": "Subscribe" }

Name Input

{ "type": "input", "inputType": "text", "placeholder": "What's your name?", "buttonText": "Send" }

Phone Number

{ "type": "input", "inputType": "tel", "placeholder": "Enter your phone number", "buttonText": "Submit" }
