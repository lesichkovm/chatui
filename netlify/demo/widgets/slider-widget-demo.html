Slider Widget Demo

Overview

The Slider Widget renders a numeric range slider for selecting values within a specified range. Users can drag the slider to select a value and submit it.

Key Features:

  • Customizable min/max values
  • Adjustable step size
  • Real-time value display
  • Default value support
  • Custom label text

Backend Response Format

To trigger a slider widget, the backend should respond with:

{ "text": "How satisfied are you with our service?", "widget": { "type": "slider", "label": "Satisfaction Rating", "min": 0, "max": 10, "step": 1, "defaultValue": 5, "buttonText": "Submit" } }

Widget Data Structure

{ "type": "slider", "label": "Select a value", "min": 0, "max": 100, "step": 1, "defaultValue": 50, "buttonText": "Submit" }

Fields:

  • type: Must be "slider"
  • label: Label text for the slider (default: "Select a value")
  • min: Minimum value (required)
  • max: Maximum value (required, must be greater than min)
  • step: Step increment (default: 1)
  • defaultValue: Initial slider position (default: midpoint)
  • buttonText: Text displayed on the submit button (default: "Submit")

Usage Examples

Satisfaction Rating (0-10)

{ "type": "slider", "label": "Rate your satisfaction", "min": 0, "max": 10, "step": 1, "defaultValue": 5, "buttonText": "Submit Rating" }

Budget Range ($0-$5000)

{ "type": "slider", "label": "Your Budget ($)", "min": 0, "max": 5000, "step": 100, "defaultValue": 2500, "buttonText": "Set Budget" }

Percentage (0-100)

{ "type": "slider", "label": "Completion Percentage", "min": 0, "max": 100, "step": 5, "defaultValue": 0, "buttonText": "Update" }
