Toggle Widget Demo

Overview

The Toggle Widget renders an on/off switch for binary choices. Users can toggle between enabled and disabled states.

Key Features:

  • Simple on/off switch
  • Default state support
  • Custom label text
  • Visual toggle animation
  • Submit button for confirmation

Backend Response Format

To trigger a toggle widget, the backend should respond with:

{ "text": "Would you like to enable notifications?", "widget": { "type": "toggle", "label": "Enable Notifications", "defaultValue": false, "buttonText": "Save" } }

Widget Data Structure

{ "type": "toggle", "label": "Enable", "defaultValue": false, "buttonText": "Submit" }

Fields:

  • type: Must be "toggle"
  • label: Label text for the toggle (default: "Enable")
  • defaultValue: Initial state (default: false)
  • buttonText: Text displayed on the submit button (default: "Submit")

Usage Examples

Enable Notifications

{ "type": "toggle", "label": "Enable Notifications", "defaultValue": false, "buttonText": "Save Settings" }

Subscribe to Newsletter

{ "type": "toggle", "label": "Subscribe to Newsletter", "defaultValue": true, "buttonText": "Update Preferences" }

Dark Mode

{ "type": "toggle", "label": "Enable Dark Mode", "defaultValue": false, "buttonText": "Apply" }
