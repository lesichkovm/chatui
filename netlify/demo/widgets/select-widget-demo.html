Select Widget Demo

Overview

The Select Widget renders a dropdown select menu with multiple options. Users can choose one option from the list, and the selection is automatically sent to the backend.

Key Features:

  • Dropdown selection with multiple options
  • Custom placeholder text for default option
  • Automatic selection handling
  • Custom styling and responsive design
  • Option validation and data mapping

Backend Response Format

To trigger a select widget, the backend should respond with:

{ "text": "Please choose your preferred department:", "widget": { "type": "select", "placeholder": "Select a department...", "options": [ { "id": "dept1", "text": "Customer Support", "value": "support" }, { "id": "dept2", "text": "Sales", "value": "sales" }, { "id": "dept3", "text": "Technical Support", "value": "tech" }, { "id": "dept4", "text": "Billing", "value": "billing" } ] } }

Widget Data Structure

{ "type": "select", "placeholder": "Select an option...", "options": [ { "id": "unique_option_id", "text": "Display text for option", "value": "backend_value" } ] }

Required Fields:

  • type: Must be "select"
  • options: Array of option objects
  • options[].id: Unique identifier for the option
  • options[].text: Text displayed in the dropdown
  • options[].value: Value sent to backend

Optional Fields:

  • placeholder: Text for the default disabled option

Usage Examples

Department Selection

{ "type": "select", "placeholder": "Choose department...", "options": [ { "id": "support", "text": "Customer Support", "value": "support" }, { "id": "sales", "text": "Sales Team", "value": "sales" }, { "id": "tech", "text": "Technical Help", "value": "technical" } ] }

Priority Level

{ "type": "select", "placeholder": "Select priority...", "options": [ { "id": "low", "text": "Low Priority", "value": "low" }, { "id": "medium", "text": "Medium Priority", "value": "medium" }, { "id": "high", "text": "High Priority", "value": "high" }, { "id": "urgent", "text": "Urgent", "value": "urgent" } ] }

Country Selection

{ "type": "select", "placeholder": "Select your country...", "options": [ { "id": "us", "text": "United States", "value": "US" }, { "id": "uk", "text": "United Kingdom", "value": "UK" }, { "id": "ca", "text": "Canada", "value": "CA" }, { "id": "au", "text": "Australia", "value": "AU" } ] }
